Faraday agents can call tools — functions that perform searches, retrieve data, or call external services. You can extend the default set with your own tools by following the patterns in this section.

## Pages in this section

- **[Built-in tools](./general-tools/)** — web search and code execution tools available out of the box.
- **[Adding a custom tool](./custom-tools/)** — tool specs, handlers, and `FaradayAgent` integration.
- **[Search tool example](./search-tool-example/)** — compact, line-numbered results for codebase search.
- **[Deployed tool example](./deployed-tool-example/)** — calling managed infrastructure from a Faraday tool.
- **[Bring your own document store](./bring-your-own-document-store/)** — connecting Faraday to a custom retrieval backend.
